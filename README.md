# TutorIA Académico - IA + Administración TI

Aplicación web de IA tipo tutor académico personalizada, desarrollada como proyecto integrador para los cursos de Inteligencia Artificial y Administración de Tecnologías de Información.

## Objetivo

Permitir que un usuario cargue materiales de estudio, documentos o audios, y converse con un tutor de IA que responde usando recuperación aumentada por generación, embeddings, búsqueda semántica y memoria conversacional básica.

## Stack propuesto

- Frontend: Next.js + TypeScript + Tailwind CSS
- Backend: Python + Django REST Framework
- IA: LangChain + LangGraph
- Base de datos: PostgreSQL + pgvector
- Infraestructura: Docker + Docker Compose
- CI/CD: GitHub Actions
- Despliegue: VPS Linux
- IaC: Terraform

## Módulos principales

- Carga de documentos
- Carga y transcripción de audio
- Procesamiento de documentos
- RAG con embeddings y vector store
- Chat tutor con memoria básica
- Generación de mini quizzes con structured output
- Panel web de documentos, conversaciones y quizzes
